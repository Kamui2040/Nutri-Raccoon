# Project overview

## Product identity

**Nutri Raccoon** is a cozy Android nutrition tracker represented by a raccoon chef mascot. Normal page headers use the app name with the mascot aligned left and Settings aligned right. The Settings page provides an About action in the top-right corner.

## Primary user experience

The home page presents meal sections such as Breakfast, Lunch, Dinner, and Snacks. Users can save recurring combinations as meal templates, add them to a day with one tap, and optionally schedule them for selected weekdays. A prominent camera action in the bottom row expands into:

1. Barcode/product scanning.
2. Meal-photo recognition.

The barcode workflow checks the local database first, then optional online sources, then offers label scanning and assisted extraction when information is missing.

## Product principles

- Reduce repetitive input without hiding uncertainty.
- Prefer confirmed label data over estimates.
- Keep personal data local by default.
- Keep the core app useful without proprietary services.
- Support realistic portions rather than only per-100-gram values.
- Use calm, non-judgmental language.
- Remain free of charge, with optional donations only.
