# Licensing map

This project uses SPDX identifiers wherever practical.

| Material | Default licence |
|---|---|
| Application source code | GPL-3.0-or-later |
| Original project documentation | CC-BY-SA-4.0 |
| Original mascot and artwork | CC-BY-SA-4.0 |
| Third-party libraries and assets | Their original compatible licences |
| External product data | The applicable database/content licence of the source |

Every third-party component must be documented before release. Generated or downloaded material must not be assumed redistributable.

The full CC BY-SA 4.0 legal text should be added before the first artwork contribution or public release. Until then, original artwork files should carry an explicit `SPDX-License-Identifier: CC-BY-SA-4.0` notice in accompanying metadata.
